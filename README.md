# Excel Outlier Analysis Project

## Project Overview
In this project, I analyzed employee beer sales data to determine whether any employees were significantly underperforming compared to projected sales.

## Analysis Performed
- Compared Actual Sales vs Projected Sales
- Used Excel Pivot Tables to calculate averages by employee
- Calculated differences between actual and projected sales
- Calculated Z-scores to identify outliers
- Identified Employee 15 as a significant outlier

## Tools Used
- Microsoft Excel
- Pivot Tables
- Z-Score Analysis
- Data Analysis

## Conclusion
Most employees were meeting projected sales, but Employee 15 consistently performed below projections across multiple shifts and appears as a statistical outlier. Employee 34 may also require monitoring.
